Introduction

Ng-ui is an open-source set of UIs blocks components designed to make building high-quality, accessible web applications easier, using angular and styled with Tailwindcss, we focuses on providing low-level UI components with an emphasis on accessibility, customization, and developer experience.

Angular Tailwindcss

Our Vision

Is to provide Angular's Developers with betters Uis Ressources such as Buttons, Cta, Hero-section, Footers, charts, pricings, dialogs, dropdowns, selects, sliders, toolbars, and many more in order to improve her development and increase her design systemes with our Components, sometimes developers are often left to build custom components, which can be a difficult and time-consuming task.

The goal of NgUI is to provide a well-funded, open-source component Code snippet that can be used to build accessible design systems, solving these implementation issues.
